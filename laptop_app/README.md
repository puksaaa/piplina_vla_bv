# Laptop app

Веб-интерфейс и FastAPI backend для управления визуальным планированием робота.

## Запуск на ноутбуке

Из корня репозитория запусти одну команду:

```powershell
powershell -ExecutionPolicy Bypass -File .\laptop_app\start.ps1
```

Скрипт сам создаёт `.venv`, устанавливает зависимости и, если его нет, создаёт
`laptop_app/.env` из шаблона. Открой затем <http://127.0.0.1:8000>.

Чтобы принудительно переустановить зависимости:

```powershell
powershell -ExecutionPolicy Bypass -File .\laptop_app\start.ps1 -Install
```

Перед запуском с реальным роботом проверь в `.env` адреса `CAMERA_BASE_URL` и
`OPENAI_BASE_URL`. Файл `.env` локальный и в Git не попадает.

## Режимы

- По умолчанию приложение строит и показывает визуальный план.
- В orchestrated mode UI передаёт команды локальному оркестратору на 4090, который
  управляет резидентным SmolVLA runner и проверяет каждый шаг по камере.

SmolVLA не является HTTP-эндпоинтом для руки. Для физического управления требуется
отдельно подготовленный и защищённый actuator/state deployment.
